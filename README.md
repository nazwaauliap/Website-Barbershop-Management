NZ Barbershop Management System
A comprehensive web-based management system designed to streamline barbershop operations. This project was developed as a Final Exam (UAS) assignment for the Web Programming course, serving as a practical application of web development fundamentals.

1. Master Data Management (8 Modules)
The system manages 8 core entities as required by the curriculum:

Barbers: Profiles of professional barbers.

Services: Catalog of hair treatments and styling.

Customers: Client database and contact management.

Categories: Classification for different types of services.

Memberships: Loyalty programs for recurring customers.

Barber Chairs: Station management within the shop.

Payment Methods: Integration of various payment gateways (Cash/Digital).

Shifts: Managing staff work schedules.

2. Transaction Modules (5 Modules)
Handles dynamic operational data:

Bookings: Appointment scheduling system.

Transactions: Main sales and service records.

Order Details: Itemized breakdown of each transaction.

Queue: Real-time customer waiting list management.

Product Sales: Retail management for hair care products.

3. Advanced Reporting (SQL Joins)
Demonstrates the ability to retrieve and display interconnected data:

Sales Performance Report: Analyzes revenue by joining transactions, barbers, and services.

Customer History: Displays detailed booking logs by joining bookings and customers.
